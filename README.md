# Dynamic Array in C++

This C++ program demonstrates the creation and manipulation of a dynamic array using `push_back` and `pop_back` operations.

## Features
- Fill the array with user input
- Display the array
- Add an element to the end of the array (`push_back`)
- Remove the last element from the array (`pop_back`)
- Dynamic memory allocation with `new` and proper deallocation with `delete[]`

## Usage
1. Compile the program with a C++ compiler (e.g., `g++ dynamic_array.cpp -o dynamic_array`)
2. Run the executable (`./dynamic_array`)
3. Enter the initial array size and elements
4. Add an element using `push_back`
5. Remove the last element using `pop_back`

# Динамічний масив на C++

Ця програма на C++ демонструє створення та роботу з динамічним масивом за допомогою операцій `push_back` і `pop_back`.

## Можливості
- Заповнення масиву з введення користувача
- Вивід масиву на екран
- Додавання елемента в кінець масиву (`push_back`)
- Видалення останнього елемента масиву (`pop_back`)
- Динамічне виділення пам'яті з `new` та правильне звільнення з `delete[]`

## Використання
1. Скомпілювати програму за допомогою компілятора C++ (наприклад, `g++ dynamic_array.cpp -o dynamic_array`)
2. Запустити виконуваний файл (`./dynamic_array`)
3. Ввести початковий розмір масиву та елементи
4. Додати елемент за допомогою `push_back`
5. Видалити останній елемент за допомогою `pop_back`
